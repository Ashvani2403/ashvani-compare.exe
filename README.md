# Ashvani Compare

A lightweight, fast desktop diff viewer for Windows built with .NET 8 and WPF.

## ✨ Features

- **Side-by-Side Comparison** - Compare two texts with synchronized scrolling
- **Character-Level Diff** - Highlights exact character changes within modified lines
- **Line Numbers** - Clear row numbering on both panels
- **Row Striping** - Alternating light backgrounds for easy line tracking
- **Color-Coded Changes**:
  - 🔴 **Red** - Removed lines
  - 🟢 **Green** - Added lines
  - 🟡 **Orange** - Modified lines (with char-level diff)
- **Blank Line Detection** - Properly detects and highlights blank line changes

## 🚀 Use Cases

- **Code Review** - Compare different versions of source code
- **Document Comparison** - Check changes between document drafts
- **Config Diff** - Spot differences in configuration files
- **Text Analysis** - Any side-by-side text comparison need


## 🛠️ Tech Stack

- .NET 8.0 (Windows)
- WPF (Windows Presentation Foundation)
- C# 12

## 📦 Installation

1. Download `AshvaniCompareSetup.exe`
2. Run the installer (per-user, no admin required)
3. Launch from Start Menu or Desktop shortcut
4. If Windows is not allowing you to install, then you can download the InstallationFiles.zip and run the app without installation.
